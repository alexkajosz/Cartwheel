Duro Security Policy

Last updated: February 8, 2026

Overview

We protect merchant data through secure transport, encryption at rest where enabled, least-privilege access, and ongoing monitoring.

Controls

  • HTTPS for data in transit.
  • Encryption at rest for stored data when enabled.
  • Access logging for sensitive actions.
  • Rate limiting on admin endpoints.
  • Backups and retention controls.

Vulnerability reporting

Please report security issues via the support email listed in the app listing.
